# dti web
a publi api that lets user retrieve images of items from game Dress to impress based on certain parameters.

## Features
- no authentication 

## Usage
```
- git clone github.com/hadeeqa/dtiweb
- python3 manage.py runserver
```
changeable in a link tier = VIP/NONVIP and count=1-10
```
port api/clothes/?Tier= & count
```
